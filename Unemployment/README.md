# Unemployment Analysis with Python

## 📌 Objective
Analyze unemployment trends using data cleaning, visualization, and insights.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📊 Features
- Data Cleaning
- Trend Analysis
- COVID-19 Impact Study
- Region-wise Analysis
- Visualization

## ▶️ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   cd src
   python analysis.py

## 📁 Output
- Graphs saved in outputs/plots
- Report saved in outputs/report.txt
